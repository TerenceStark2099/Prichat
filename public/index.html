Why Prichat is Different

End-to-End Encryption

All messages are encrypted client-side using a unique AES key derived from the room code. The server only sees ciphertext.

Ephemeral Rooms

The entire room and its history are deleted from the database moments after the last participant closes the chat or the browser tab.

Zero Authentication

No email, no phone number, no cookies. Just enter a code and start talking. Your identity remains fully private.

How It Works in 3 Steps

1.

Generate or Enter a Code

The room code (e.g., H8Q3YZ) is used as the secret password to generate the encryption key.

2.

Chat with E2E Security

Every message is encrypted in your browser before it ever touches the server. Only users with the room code can decrypt it.

3.

The Room Vanishes

When everyone leaves the chat (by closing the tab or hitting 'Leave Room'), the room and all messages are immediately purged from the database.
